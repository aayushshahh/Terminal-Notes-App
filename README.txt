# Terminal-Notes-App
A Simple Terminal Based Note Taking App Built using Node.js and amp Modules. First ever Project on Node.js
Run the App in Terminal with the Following Commande : node App.js
The App provides four Commands : 1. Add a Note (Add)
                                 2. Delete a Note  (Remove)
                                 3. List All Note Titles (List)
                                 4. Read a Note (Read)
The Add Command Requires a Title and a Body
The Remove Command Requires a Title
The Read Command requires a Title
node App.js --help - for Basic Info
node App.js Add --title="Note Title" --body="Note Body"
node App.js Remove --title="Note Title"
node App.js List
node App.js Read --title="NOte Title"
